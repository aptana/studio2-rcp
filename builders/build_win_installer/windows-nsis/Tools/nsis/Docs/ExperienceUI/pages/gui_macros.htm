GUI manipulation macros

XPUI_FUNCTION_GUIINIT - DEPRECIATED - You should not insert this macro anymore, as it is inserted automatically now. In pre-1.1M2 versions of the ExperienceUI, this macro initialized the visual style on the installer.

XPUI_HEADER_TEXT header_text header_subtext - Sets the text on the installer header to whatever you specify.
