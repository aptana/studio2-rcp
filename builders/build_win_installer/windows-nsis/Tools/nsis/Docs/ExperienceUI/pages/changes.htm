Changes in version 1.1

The ExperienceUI now comes standard with a Language Selection page, InstallOptions automization, and many new graphical enhancements. Plus, native WAnsis support and the ability to convert Modern UI, UltraModern UI, and InstallSpider UI scripts make the ExperienceUI a must for any NSIS developer.

Changes in version 1.08

  • Nav Bar, Tutorial, and Dynamic Browser in readme
  • New Left Logo by ZmAn3
  • The "Left Branding Image" is now the "Left Logo"
  • Various readme layout changes

Changes in version 1.07

Version 1.07 of the ExperienceUI did not contain any new features, only bugfixes. For more information, please see the Version History.
